# Design Prompt Library Seed Content

> 작성일: 2026-03-29

## 1. 카드 작성 규칙

모든 카드는 아래 순서를 유지한다.

1. `Title`
2. `Short Summary`
3. `Visual Traits`
4. `Do`
5. `Avoid`
6. `Stitch Prompt`
7. `Suggested Outputs`

프롬프트 원칙:
- 한 문장 안에 너무 많은 미감을 섞지 않는다.
- 반드시 `layout`, `typography`, `color`, `motion`, `imagery`, `tone` 중 최소 4개 이상을 명시한다.
- "beautiful", "modern", "cool" 같은 비어 있는 형용사는 가급적 제거한다.

## 6. 카테고리 운영 메모

- 초기 홈에는 20개 전부 노출하지 말고 `featured 6 + new 4 + collections 4`로 압축한다.
- 카테고리 상세는 이미지보다 프롬프트와 trait 토글이 먼저 보여야 한다.
- 검색 키워드는 브랜드명과 추상화 이름을 함께 지원한다.
  - 예: `apple`, `quiet premium`, `clean hierarchy`, `enterprise grid`
- 추후엔 각 카드마다 `lighter`, `bolder`, `more motion`, `more premium` 파생 프롬프트를 추가한다.
